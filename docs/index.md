# PawnPro Debugger

Debugger visual (DAP) para a linguagem **Pawn** (SA-MP / open.mp), integrado à
extensão [PawnPro](https://github.com/NullSablex/PawnPro), para um servidor local
de desenvolvimento.

Permite depurar código Pawn direto no editor: breakpoints (simples, condicionais
e por contagem de acertos), logpoints, step, inspeção e edição de variáveis, e
**pausar na linha de um erro de runtime** antes de a VM abortar — em SA-MP e
open.mp.
